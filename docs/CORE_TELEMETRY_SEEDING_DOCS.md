# Сидинг core-телеметрии

## Назначение

Документ описывает reduced-модель сидинга, реализованную в `generate_core_synthetic_telemetry.py`.

Эта версия намеренно фокусируется на малом наборе сырых сигналов локомотива:

- Скорость
- Коэффициент сцепления
- Расход и уровень топлива
- Ключевые давления
- Ключевые температуры
- Напряжение и ток
- Сырые коды неисправностей

Не генерирует:

- GPS
- Heading
- Ускорение как выходную колонку
- Напряжение батареи
- Телеметрию связи
- Health index
- Таблицы деривированных алертов
- Таблицы диспетчерских команд

## Выходные файлы

Скрипт записывает один датасет:

- `synthetic_output_core/telemetry.csv`
- `synthetic_output_core/telemetry.jsonl`
- Опционально: `synthetic_output_core/telemetry.parquet`

## Локомотивы

| # | ID | Тип |
| --- | --- | --- |
| 1 | `KZ8A-001` | `KZ8A` (электрический) |
| 2 | `TE33A-002` | `TE33A` (дизель-электрический) |
| 3 | `KZ8A-003` | `KZ8A` (электрический) |

Гарантируется минимум один электрический и один дизель-электрический.

## Детерминизм

- Глобальный seed: `RANDOM_SEED = 42`
- Per-locomotive seed: `RANDOM_SEED + index × 10_000`

Повторные запуски дают идентичную телеметрию.

## Скрытая модель состояния

Генератор использует компактную hidden-state модель. Скрытое состояние не записывается в датасет, но управляет всеми выходными колонками.

### Фазы работы

Каждый локомотив циклически проходит:

1. `idle` — холостой ход
2. `accel` — ускорение
3. `cruise` — крейсерский ход
4. `brake` — торможение
5. `stop` — остановка

### Скрытые драйверы

- Целевая скорость
- Положительное ускорение
- Тормозной спрос
- Сигнал нагрузки
- Термическая инерция

## Связи между параметрами

Выходные колонки не являются независимыми случайными значениями.

### 1. Скорость

- `speed_kmh` генерируется по фазе и сглаживается
- `accel` постепенно поднимает скорость
- `cruise` удерживает скорость в полосе с малой осцилляцией
- `brake` постепенно снижает скорость
- `stop`/`idle` держат скорость ~0

### 2. Ток тяги

- `traction_current_a` — низкий при остановке
- Резко растёт при ускорении
- Умеренный при крейсерском ходе
- Падает при торможении
- Sustained высокий ток повышает температуры
- Низкий `adhesion_coeff` снижает эффективный ток при том же спросе

### 3. Давления тормозов

- `brake_cylinder_pressure_bar` растёт при тормозном спросе
- `brake_pipe_pressure_bar` падает при росте давления цилиндра
- Изменения давления согласованы с падением скорости
- Слабое сцепление может снизить тормозную эффективность

### 4. Сцепление

- `adhesion_coeff` — качество сцепления колесо-рельс
- Меняется медленно, не как белый шум
- Обычно в здоровой полосе, иногда проваливается на sustained окно
- Низкое сцепление ослабляет тяговый отклик и торможение на скорости

### 5. Температуры

- `traction_motor_temp_c` — быстрая реакция на sustained нагрузку
- `bearing_temp_c` — медленнее и с меньшей амплитудой
- `transformer_temp_c` (KZ8A) — следует за sustained электрической нагрузкой
- `coolant_temp_c` (TE33A) — следует за sustained дизельной нагрузкой
- Температуры постепенно охлаждаются при падении нагрузки

### 6. KZ8A электрические правила

- `catenary_voltage_kv` — стабилен с умеренным шумом
- Высокая нагрузка вызывает mild voltage sag
- Sustained высокая нагрузка повышает `transformer_temp_c`

### 7. TE33A правила топлива и давления

- `fuel_rate_lph` растёт с нагрузкой
- `fuel_level_l` — кумулятивный инверс расхода, только убывает
- `oil_pressure_bar` — в нормальной полосе при здоровой работе
- Термический/аномальный стресс может снизить давление масла

### 8. Правила неисправностей

- `fault_code` — null большую часть времени
- Становится non-null только при аномальных окнах
- Неисправности — sustained окна, не одиночные сэмплы

## Инжектированные аномалии

Генератор модифицирует raw-телеметрию напрямую во время аномальных окон.

### Общая аномалия

**`BRAKE_RESPONSE_WEAK`** — давление цилиндра слабее ожидаемого при торможении, давление магистрали остаётся слишком высоким. Создаёт реалистичную основу для аларма торможения на скорости.

### KZ8A аномалии

- **`KZ8A_VOLTAGE_DIP`** — напряжение контактной сети падает ниже нормы
- **`KZ8A_TRANSFORMER_HOT`** — температура трансформатора растёт выше тренда

### TE33A аномалии

- **`TE33A_OIL_LOW`** — давление масла падает ниже нормы
- **`TE33A_COOLANT_HOT`** — температура coolant растёт, расход топлива может увеличиться

## Схема

| Колонка | Тип | Nullable | Тип лок. | Описание |
| --- | --- | --- | --- | --- |
| `timestamp` | `TIMESTAMP(3)` | Нет | все | Время события (мс точность) |
| `t_ms` | `BIGINT` | Нет | все | Относительное время (мс от старта) |
| `locomotive_id` | `VARCHAR(32)` | Нет | все | Идентификатор локомотива |
| `locomotive_type` | `VARCHAR(16)` | Нет | все | `KZ8A` или `TE33A` |
| `speed_kmh` | `REAL` | Нет | все | Скорость поезда |
| `adhesion_coeff` | `REAL` | Нет | все | Коэффициент сцепления колесо-рельс |
| `traction_current_a` | `REAL` | Нет | все | Ток тяги |
| `brake_pipe_pressure_bar` | `REAL` | Нет | все | Давление тормозной магистрали |
| `brake_cylinder_pressure_bar` | `REAL` | Нет | все | Давление тормозного цилиндра |
| `traction_motor_temp_c` | `REAL` | Нет | все | Температура тягового мотора |
| `bearing_temp_c` | `REAL` | Нет | все | Температура подшипника |
| `fault_code` | `VARCHAR(64)` | Да | все | Код неисправности |
| `catenary_voltage_kv` | `REAL` | Да | KZ8A | Напряжение контактной сети |
| `transformer_temp_c` | `REAL` | Да | KZ8A | Температура трансформатора |
| `fuel_level_l` | `REAL` | Да | TE33A | Уровень топлива |
| `fuel_rate_lph` | `REAL` | Да | TE33A | Расход топлива |
| `oil_pressure_bar` | `REAL` | Да | TE33A | Давление масла |
| `coolant_temp_c` | `REAL` | Да | TE33A | Температура охлаждающей жидкости |

## Семантика null

Type-specific числовые поля генерируются как `np.nan` когда не применимы:

- JSONL → `null`
- CSV → пустые ячейки
- Семантика: «не применимо», не «ноль»

## Рекомендуемая raw-таблица

```sql
CREATE TABLE telemetry_raw_core (
    timestamp TIMESTAMPTZ NOT NULL,
    t_ms BIGINT NOT NULL,
    locomotive_id VARCHAR(32) NOT NULL,
    locomotive_type VARCHAR(16) NOT NULL,
    speed_kmh REAL NOT NULL,
    adhesion_coeff REAL NOT NULL,
    traction_current_a REAL NOT NULL,
    brake_pipe_pressure_bar REAL NOT NULL,
    brake_cylinder_pressure_bar REAL NOT NULL,
    traction_motor_temp_c REAL NOT NULL,
    bearing_temp_c REAL NOT NULL,
    fault_code VARCHAR(64) NULL,
    catenary_voltage_kv REAL NULL,
    transformer_temp_c REAL NULL,
    fuel_level_l REAL NULL,
    fuel_rate_lph REAL NULL,
    oil_pressure_bar REAL NULL,
    coolant_temp_c REAL NULL,
    PRIMARY KEY (locomotive_id, timestamp)
);
```
